## Objetivo del sistema

Desarrollar un Sistema de Gestión de Pedidos para una Lavandería que permita registrar, consultar, actualizar y administrar los pedidos de los clientes mediante una aplicación de consola desarrollada en Python, aplicando los conocimientos adquiridos en la asignatura de Lógica de Programación.

---

## Descripción de funcionalidades

El sistema ofrece las siguientes funcionalidades:

- Registrar nuevos pedidos.
- Consultar pedidos mediante su número.
- Mostrar todos los pedidos registrados.
- Actualizar el estado de un pedido.
- Aplicar un descuento del 10 % a pedidos con diez o más prendas.
- Eliminar pedidos registrados.
- Mostrar un resumen de pedidos y del total recaudado.
- Consultar las tarifas de los servicios disponibles.
- Buscar pedidos por el nombre del cliente.
